# CumbreGanadera

##Descripción Breve
Nuestro sistema se centrara en la gestion de haciendas con todos sus aspectos primordiales, ademas de la venta de productos de cada hacienda

## Tecnologías utilizadas
- ASP.NET Web Forms
- SQL Server
- Bootstrap
